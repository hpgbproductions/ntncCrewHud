# ntncCrewHud

Simple HUD overlay for Train Crew.

![](img/s1.png)

![](img/s2.png)

Download from the Releases section on the right.

To use it, run `ntncCrewHud.exe`. while in the diagram, press U to hide the driving UI. The plugin is blue when the train passes the next station. It turns orange when you need to stop at the next station.

The file `settings.txt` contains some basic options. You can hide the items that you don't want, and change the size of the stopping guide. The file is only loaded when the program is started.
